All processing is performed **locally**, ensuring low latency and privacy preservation.

This project demonstrates how deep learning‚Äìbased **face analytics** can be applied in real-time scenarios.

The focus of this project is on:
- Real-time performance
- CPU-based inference
- Practical deep learning pipeline design

## üöÄ Features

- **Multi-Task Learning**  
  Simultaneous prediction of age, gender, and emotion from a single face image.

- **Real-Time Inference**  
  Optimized for real-time execution on CPU.

- **Privacy-First Design**  
  All computations are performed locally; no cloud processing involved.

- **Modular Pipeline**  
  Independent stages for face detection and attribute prediction.
